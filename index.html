Programming & TileMill 101

What do we mean by "programming"?

  • Telling a computer what to do
  • A program is just a text file in a language that a computer understands
  • Other programs interpret these files to tell the hardware what to do

Let's start with HTML

  1. You have some text which has some implicit structure, e.g. headings, paragraphs, lists
  2. You write an HTML file using a particular syntax
  3. The HTML adds explicit structure
  4. Your web browser understands the HTML, and displays your content with the structure you define
  5. Let's have a look

Making it Shiny

  1. HTML gives structure, but it doesn't dictate what things look like
  2. Your web browser reads a CSS file to decide how to style things
  3. CSS lets you style many HTML elements at once using tags & classes

Lesson 2
Working with geodata

What is a map?

A map does not just chart, it unlocks and formulates meaning; it forms bridges between here and there, between disparate ideas that we did not know were previously connected.

― Reif Larsen, The Selected Works of T.S. Spivet

What is a map?

  • Maps are not always for navigation.
  • At its core, a map is a visualisation of information.

    • What makes a map good?

    Map-making had never been a precise art on the Discworld. People tended to start off with good intentions and then get so carried away with the spouting whales, monsters, waves and other twiddly bits of cartographic furniture that the often forgot to put the boring mountains and rivers in at all.

    ― Terry Pratchett, Moving Pictures

    What makes a map good?

    • Figure out what you're saying, and say only that.
    • Don't cram everything you think might be relevant or helpful onto the map.
    • If you're showing data about provincial dialects, does the viewer need to know about train stations?
    • If you're showing data about transport accessibility, does the viewer need province boundaries?

    What does "geodata" mean?

    • Geodata has coordinates, not just place names
    • A spreadsheet with city names, country names, or addresses is not geodata (the way we mean it)
    • A spreadsheet with latitude & longitude fields is

    What does "geodata" mean?

    • There are a few geo-specific file types you will find:
    • CSV/XLS: if it has lat/lon fields. Can only contain points.
    • Shapefile: Common "standard" for geospatial data. Can be annoying to work with.
    • KML/KMZ: Very common for crowdsourced data.
    • There are plenty more (PostGIS, SpatialLite, GeoJSON...) but they're a bit exotic.

    Can we see some?

    Sure.

    Compare the Martial Arts Clubs in Victoria dataset to the Location of Centrelink Offices dataset.

    • They're both CSV files...
    • They're not both geodata!
